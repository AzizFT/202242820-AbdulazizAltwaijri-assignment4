# AI Usage Report - Assignment 4

## 1. Tools Used

The main AI tool used in this assignment was ChatGPT.

## 2. Use Cases

AI was used for:

- planning the final portfolio structure
- reviewing Assignment 4 requirements
- improving documentation
- suggesting presentation structure
- refining JavaScript logic
- improving user experience explanations
- brainstorming a more useful and creative feature for the final portfolio
- designing the Career Path Recommender logic
- improving how the feature connects to innovation and user experience

## 3. Example Prompts Used

- How can I turn my Assignment 3 portfolio into a final polished Assignment 4 project?
- What should a final portfolio technical documentation file include?
- Help me write a user-friendly README for a web portfolio.
- How should I structure a 5 to 7 minute presentation for this project?
- What are good future improvements for a portfolio web application?
- Suggest a useful creative feature for a software engineering portfolio.
- How can I build a simple recommendation feature using JavaScript and localStorage?

## 4. Benefits

AI helped me:

- organize the final assignment requirements
- improve the quality of my documentation
- connect my features to the grading rubric
- prepare a clearer presentation plan
- reduce time spent planning repetitive sections
- think of a more useful creative feature than a simple progress checklist

## 5. Challenges

Some AI suggestions were too broad, so I had to simplify them. I also had to make sure the final project stayed realistic and matched my own portfolio instead of becoming too complicated.

Another challenge was avoiding generic documentation. I reviewed and adjusted the AI-generated suggestions so the final files matched my actual project features.

## 6. Risks of AI Use

Risks included:

- relying too much on generated explanations
- accepting code without understanding it
- adding unnecessary features
- writing documentation that sounds too generic
- leaving unused CSS or JavaScript in the project

## 7. Responsible Use

I reviewed and edited AI suggestions before using them. I tested the website manually and made sure the final work matched my project structure and assignment requirements.

I also removed or replaced suggestions that did not add real value to the portfolio, such as a simple progress checklist. Instead, I used a Career Path Recommender because it is more useful and easier to explain during the presentation.

## 8. Learning Outcomes

Through this assignment, I improved my understanding of:

- final project polishing
- documentation quality
- API usage
- localStorage state management
- presentation planning
- professional web application structure
- creating a recommendation feature using multiple user choices
- using conditional logic and localStorage together

## 9. Final Reflection

AI was useful as a planning and review tool, but the final decisions, edits, testing, and submission were completed manually to keep the work accurate and responsible.
