# Assignment 3 - Advanced Portfolio Website

This project is an improved version of my portfolio website. It includes API integration, advanced logic, state management, and improved user interaction.

## Features
- Responsive design
- Theme toggle with saved preference
- Visitor name saving
- Project search
- Project filtering
- Project sorting
- Contact form validation
- GitHub API integration
- Error and success messages

## How to Use
1. Enter your name to personalize the page.
2. Browse projects using search, filter, and sort options.
3. View GitHub repositories.
4. Use the contact form.
5. Toggle theme.

## Technologies
- HTML
- CSS
- JavaScript
- GitHub API
- localStorage

## Setup
1. Clone the repository
2. Open index.html
3. Ensure internet connection for API
