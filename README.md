# Assignment 4 - Final Personal Portfolio Web Application

This is the final version of my personal portfolio web application. It brings together the work from previous assignments into a polished and responsive web application with API integration, state management, interactive project browsing, a career recommendation feature, form validation, and professional documentation.

## Live Demo

https://azizft.github.io/202242820-AbdulazizAltwaijri-assignment4/

## Features

- Responsive design for desktop, tablet, and mobile
- Light/Dark theme toggle
- Saved theme preference using localStorage
- Visitor name personalization
- Project search, filtering, and sorting
- GitHub API integration
- Career Path Recommender based on interest and skill level
- Contact form validation
- Helpful success and error messages
- Clean final UI
- Presentation folder for slides and demo video

## How to Use

1. Open the website.
2. Enter your name in the personalization section.
3. Use the Projects section to search, filter, and sort projects.
4. Use the Career Path Recommender to choose an interest and level, then receive a personalized learning direction.
5. View live GitHub repositories loaded from the GitHub API.
6. Use the Contact form and follow validation messages.
7. Toggle the theme using the theme button.

## Setup Instructions

1. Clone or download this repository.
2. Open the project folder.
3. Open index.html in a browser.
